En esta aplicación se muestran dos mapas:


  • En el primero se muestran las farmacias de la comunidad de Madrid. Si se mantiene el cursor sobre ellas, se obtiene su calle y su horario de apertura al público.
  • En el segundo se muestran distintos lugares que visitar en Madrid (Museos, parques, discotecas, teatros...). Si se mantiene el cursor sobre ellos, se obtiene su nombre.

ACLARACIÓN:

Debido a que no tenemos las coordenadas de los lugares, pero si las direcciones, se ha utilizado la API de Google Maps para obtenerlas a partir de éstas. La API tiene un límite diario de consultas si no se obtiene el servicio premium, por lo que sólo se muestran algunos resultados. Lamento las molestias.
