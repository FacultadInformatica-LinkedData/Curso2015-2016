Analyse Data Set

Farmacias de guardia

Columna Tipo Rango Comentarios Problemas
Localidad String 6 caracteres No nulo Mismo valor en todas las filas. Columna desechable.
Barrio String [6-29] caracteres No nulo
Fecha Date ['2015-10-01'-'2015-10-31'] No nulo
Farmacia String [1-5] cifras No nulo. Aunque son valores numéricos, son IDs, por lo que se deben tratar como Strings.
Dirección String [12-81] caracteres No nulo
Duración Choice (5 alternativas) [30-35] caracteres No nulo Se deberían sustituir por IDs más claros para las alternativas.
Con strings tan largos se pueden producir errores al poner una alternativa no válida.
Teléfono String 7 cifras Puede ser nulo Recordar añadir el prefijo 91 de Madrid.
